# Portfolio de Ciencia de Datos — Brandon Uriel Garcia Sanchez

Repositorio centralizado con mis proyectos de ciencia de datos aplicada a negocio, finanzas e IA. Estudiante de Ciencia de Datos, actualmente interno en HSBC GSC.

## Estructura

- **`proyectos/`** — análisis completos, cada uno en su propia carpeta con datos, notebook y README propio.
- **`scripts/`** — ejercicios de curso, prácticas y apuntes sueltos (no son proyectos terminados con conclusión propia).
