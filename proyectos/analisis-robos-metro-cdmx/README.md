# Análisis de Robos en el Metro de la CDMX

Limpieza y análisis exploratorio de datos de incidentes delictivos reportados por la Fiscalía de la Ciudad de México (2020-2025) en el sistema de Metro, con un modelo de regresión lineal para identificar tendencias y un dashboard geográfico de los incidentes.

**Datos:** reportes de la Fiscalía CDMX 2020-2025 (csv/xlsx), coordenadas de incidentes (kmz).
**Herramientas:** Python (pandas, regresión lineal), Power BI.
